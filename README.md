# Cook And Laugh / Варити І Реготати

## Prerequisites
You have to have installed:
* Docker
* JVM 22

## Run application
Execute command:

```shell
./mvnw spring-boot:run
```

## Tests
To run tests execute command:

```shell
./mvnw test
```

## Swagger
To open swagger go to
`http://localhost:8080/swagger-ui/index.html`
